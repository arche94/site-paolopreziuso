## Goal
Make the page wrapper, Nav, and Hero components fully responsive across mobile (375px), tablet (768px), desktop (1280px), and 4K (1920px+).

## Purpose
Phase 3 goal is a pixel-correct, responsive site ready to share publicly. The wrapper, nav, and hero are the above-the-fold experience — these must work on mobile before the rest matters.

## Output
- `.doc` wrapper: responsive padding (tight on mobile, full on desktop)
- Nav: hamburger menu on mobile, collapses to icon, expands inline links on md+
- Hero: fluid h1 (scales from ~44px mobile to 88px desktop), hidden watermark on mobile, responsive spacing
</objective>

<task type="auto">
  <name>Task 3: Hero fluid headline and responsive spacing</name>
  <files>src/components/Hero.astro</files>
  <action>
    The Hero has a fixed 88px h1 and 96px top padding — both too large for mobile.

    Changes needed:

    1. **h1 font size** — replace the scoped `font-size: 88px` with a CSS clamp in the scoped style:
       `font-size: clamp(44px, 9vw, 88px);`
       This scales fluidly from 44px (375px viewport) to 88px (desktop).

    2. **Top padding** — replace `padding: 96px 0 72px` with:
       `padding: clamp(48px, 8vw, 96px) 0 clamp(40px, 6vw, 72px);`

    3. **Watermark (`.mark-big`)** — hide on viewports below md breakpoint. Add to scoped style:
       ```css
       @media (max-width: 767px) {
         .mark-big { display: none; }
       }
       ```

    4. **Lede max-width** — change `max-width: 620px` to `max-width: min(620px, 100%)` so it doesn't overflow on small screens.

    5. **`.doc-dateline` gap** — reduce gap on mobile. Add:
       ```css
       @media (max-width: 767px) {
         .doc-dateline { gap: 12px; flex-wrap: wrap; }
       }
       ```

    Avoid: changing font families, colors, letter-spacing — design tokens stay intact.
    Avoid: adding Tailwind utility classes inside the hero scoped style — use CSS media queries here for precision.
  </action>
  <verify>At 375px: h1 renders 44-48px, no horizontal overflow, watermark invisible, hero top padding ~48px. At 1280px: h1 still ~88px, watermark visible.</verify>
