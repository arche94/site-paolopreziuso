<objective>
## Goal
Scaffold a working Astro project with static output configured, verify it builds and runs locally via a Playwright smoke test.

## Purpose
Establishes the project foundation before any components or content are added. Phase 2 depends on a building Astro project existing. Static output (no adapter) is correct for this portfolio — Cloudflare Pages natively serves static files from `dist/`. Playwright is added now so Phase 2 component verification is automated from the start.

- `package.json` with Astro + Playwright devDependencies
- `astro.config.mjs` configured for static output
- `tsconfig.json` extending Astro's base config
- `.gitignore` covering `dist/`, `node_modules/`, `.astro/`
- `src/env.d.ts` with Astro client types
- `src/pages/index.astro` — minimal placeholder page
- `src/layouts/Base.astro` — minimal HTML shell layout
- `playwright.config.ts` — configured against local dev server
- `tests/smoke.spec.ts` — smoke test asserting page loads correctly
